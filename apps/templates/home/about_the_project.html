About the Project

This dashboard was developed as a data visualization and forecasting tool to analyze Alberta’s crude oil production trends. Built using Microsoft Visual Studio, the project utilizes Python as its primary language, complemented by HTML, CSS, JavaScript, Bootstrap, Gitand Plotly for visualizations. The early development phase also included experimentation with MATLAB, later transitioning to a fully web-based framework.

Real-time data is sourced from the Government of Alberta’s public API, ensuring up-to-date and reliable insights. The project involved extracting raw production data directly from these APIs, followed by extensive data wrangling and time-series calculations to generate meaningful performance metrics and trend visualizations. Custom logic was implemented to compute key indicators such as year-over-year changes, monthly growth rates, cumulative production, forecasting models, and correlation matrices, all dynamically rendered through interactive Plotly charts.

The layout follows a clean, modular design aligned with modern data storytelling principles, emphasizing usability, clarity, and policy relevance.
