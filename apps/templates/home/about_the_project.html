About the Project

This dashboard was developed as a data visualization and forecasting tool to analyze Alberta’s oil production trends. Built using Microsoft Visual Studio, the project utilizes Python as its primary language, complemented by HTML, CSS, SCSS, JavaScript, Bootstrap and Plotly for visualizations. Early development also included experimentation with MATLAB, later transitioning to a fully interactive web-based framework.

Real-time data is sourced from the Government of Alberta’s public API, ensuring up-to-date and reliable insights. The dashboard features production KPIs, correlation matrices, forecasting models (like Simple Linear Regression and ARIMA), and industry-specific visuals like the Exponential Decline Curve.

The layout follows a clean, modular design aligned with modern data storytelling principles, emphasizing usability, clarity, and policy relevance.
